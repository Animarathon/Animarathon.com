Buy a Projector Cycle Unit!

What is a Cycle Unit?

A Cycle Unit is one minute of screen time per advertising cycle. Cycle Units loop, and are on multiple projectors, so a Cycle Unit is actually much longer then one minute of screen time.

When you order more than one Cycle Unit, your minutes will be staggered by default to increase effectiveness, unless you request that they be shown continuously instead.

Q: What do I need to provide?

A: Partners should email a 1024x768 .png file by March 20th. Please contact us if you have any questions.

Q: Do you have any design guidelines?
