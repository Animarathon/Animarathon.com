ANO Apps

Anime In Northwest Ohio maintains a curated list of tools and software it recommends.

Please note that ANO doesn't actually make or own any of these tools, some of it's members just recommend them.

Writing & Fanfiction

Libreoffice

Libreoffice is a fully featured office suite that is perfect for general use. Libreoffice is free software, making it an excellent choice for students.

Art and Design

Krita

Krita is a digital painting program that allows you to draw using your computer. Krita is free software, making it an excellent choice for students.

Inkscape

Inkscape is a vector graphics editor that is great for making certain styles of artwork.

Blender

Blender is a feature packed 3D modeling tool that is extremely powerful.

GNU Image Manipulation Program

The GNU Image Manipulation Program is a fully featured image editor suite and is useful for many purposes.

Synfig Studio

Synfig Sudio is a multiplatform 2D animation suite.

Game Development

Atom

Atom is a powerful general purpose text editor that is free software.

Brackets

Brackets is a well designed text editor that is great for working on code for the web.

Ren'Py

Ren'Py is a 2D Visual Novel game engine that makes game creation easy and simple. Ren'Py uses python, and can be scripted in a way that is very close to English.

Godot

Godot is a fully featured conventional game esngine that can be used to create 2D and 3D games.

Fedora

Fedora is a free operating system that is useful for many kinds of development.

Advanced Cosplay

Conductive Thread, Fiber Optics, Conductive Ink, Wearable Computers, and Thermochromatic Paints are powerful and underused tools in the cosplayer's toolbox.

The Arduino IDE can be used to program many wearable computing platforms, useful for making smart and interactive cosplay pieces.
